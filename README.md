# Antonio Guerrero — Data & Analytics Engineering Portfolio

## Overview

This portfolio documents a progression from analytics foundations into data engineering, analytics engineering, fintech data systems, graph auditability, cloud data platforms, and ML/AI-enabled operational systems.

The central portfolio narrative is the **Lexian Engineering Portfolio**: a fictional fintech engineering ecosystem built with synthetic data only. Lexian is not a real company and does not contain real customer, employer, Klar internal, or confidential information.

## How To Review This Portfolio

1. Start with [Lexian Transaction Engine](https://github.com/arza1uz/lexian-transaction-engine).
2. Review the 12-month engineering milestones below.
3. Review analytics foundation projects for SQL, cleaning, and visualization.
4. Treat Planned work as roadmap direction, not completed claims.

## Portfolio Philosophy

This portfolio is organized around business problems, not random technologies. Each project introduces tools when the business problem requires them.

Each milestone should produce at least one of the following:

- a portfolio project
- a professional credential
- a LinkedIn-relevant skill
- a technical competency

The Lexian Engineering Portfolio should remain a set of focused repositories, not a giant monorepo.
